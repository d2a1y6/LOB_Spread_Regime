### 1.1 基金风格飘移如何产生

+ 多任务学习识别基金风格飘移的动机（主任务“下一期是否发生重度漂移”，辅助任务“动机属于哪类”）
  + 多任务学习：分为共享层（Shared Layer）和任务私有层（Task-Specific Layer），让一个模型同时学习多个相关任务，通过任务间的信息共享来互相促进，最终让每个任务的性能都比单独训练时更好（具体原理尚不明确）
  + 手动规定几类风格飘移动机，例如：① 策略动机特征（基金前 3 期行业轮动收益、风格调整幅度）；② 市场适配特征（市场风格切换幅度、行业景气度匹配度）；③ 经理能力特征（经理过往漂移后的业绩修复率、选股胜率），通过多任务学习框架分类飘移行为。可以继续探究各类飘移动机对业绩的影响
  + **创新点：没人用机器学习做过这个；问题：也没那么新。我觉得不适合作为侧重方向**
+ ==基于图神经网络的漂移传导路径识别==
  + 节点包括 “基金节点”（属性：规模、风格、经理特征）、“股票节点”（属性：市值、估值、行业）、“行业节点”（属性：景气度、资金流入），边权重为 “基金持仓比例”“股票所属行业权重”；采用 GraphSAGE 模型，通过邻居节点特征聚合（如某基金持仓的 10 只股票中 8 只因行业下跌被动调整风格），计算 “行业风格冲击→股票风格变化→基金漂移” 的传导强度；对比不同市场环境下的传导效率（如牛市中行业资金拥挤导致传导效率提升 30%，熊市中传导效率下降 15%）；
  + **这个方向本身撑得起一篇论文的聚焦点，困难在数据是否可得**

### 1.2 基金风格飘移有何影响

+ 基于时序注意力机制的漂移效应动态分解

  + 时序时序注意力模型（Temporal Attention）是一种结合了时序数据特性和注意力机制的模型组件，常被用于增强循环神经网络（如 LSTM、GRU）对时序数据的处理能力。它的核心思想是：在处理序列数据（如时间序列、文本、语音）时，让模型 “自动关注序列中更重要的时刻或位置”，从而提升对关键信息的捕捉能力

  + 以 “基金月度漂移程度、市场环境、业绩数据” 为时序输入，通过注意力权重热力图，分解漂移效应为 “短期冲击效应”（漂移后 1 个月，效应系数 0.23）、“中期调整效应”（漂移后 2-3 个月，效应系数 - 0.18）、“长期稳态效应”（漂移后 4-6 个月，效应系数 0.05），动态捕捉漂移对业绩的 “短期冲击 - 中期调整 - 长期稳态” 效应，量化不同阶段的效应差异
  + **挺新的，就是不知道是否能做出结果**

+ ==基于因果森林（Causal Forest）的漂移效应异质性挖掘==

  + 因果森林（Causal Forest）是一种结合了随机森林和因果推断的机器学习方法，专门用于估计个体处理效应（Individual Treatment Effect, ITE），简单说就是回答 “对某个个体而言，做某件事（处理）和不做（对照）的结果差异有多大”
  + 处理变量为 “是否发生漂移”，结果变量为 “下季度超额收益率”，混淆变量为基金规模、经理任职年限、机构持有比例等。**本质是更复杂的因果识别**，可以识别出特定特征组合出的“子群体”，e.g., “小规模+ 高机构持有”漂移后超额收益平均提升大

  + **这个题可以找lsy。巨大优点是数据几乎都是截面的，非常好获得**

## 方向2：基于PSM识别风格飘移-业绩表现的因果效应

+ 创新点：既往研究虽多针对风格飘移对业绩表现的影响，但从未良好解决二者间的反向因果问题。通过PSM构造准实验可以优化这一点
